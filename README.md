# Titanic Survival Prediction

A Machine Learning project built using Python and Scikit-learn to predict whether a Titanic passenger would survive.

---

## Features

- Data Cleaning
- Feature Engineering
- Label Encoding
- One-Hot Encoding
- Random Forest Classifier
- GridSearchCV Hyperparameter Tuning
- Cross Validation
- Model Saving & Loading
- Passenger Survival Prediction
- Survival Probability Prediction

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
